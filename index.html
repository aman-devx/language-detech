How I Built This App

Frontend Stack

Built with vanilla HTML5 and JavaScript. Styling is handled via Tailwind CSS for a professional, responsive layout without the overhead of a framework.

ML Architecture

The app communicates with a specialized Flask-based API hosted on Render. It uses a trained Machine Learning model to analyze linguistic patterns and return predictions.

The Implementation Flow

  1. Data Acquisition: Text is sanitized and captured via the input interface.
  2. Asynchronous Request: The JavaScript fetch API sends a POST request to the /predict endpoint with the payload.
  3. API Processing: The remote server vectorizes the text and processes it through an ML pipeline.
  4. UI Update: The results are parsed and dynamically rendered into the DOM with smooth CSS transitions.
